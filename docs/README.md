# docs/

Durable, institutional documentation — architecture decisions, governance policy, and lessons learned, preserved independently of individual contributors.

| Folder | Purpose |
|---|---|
| `architecture/` | Architecture decision records and diagrams |
| `governance/` | Governance and engineering policy documents |
| `lessons-learned/` | Captured lessons from each Roadmap phase |

See Section 16 (Solution-Oriented Engineering) and Section 21 (Repository Structure) of the root README for the Repository-First Engineering and Institutional Knowledge Preservation principles this directory implements.
