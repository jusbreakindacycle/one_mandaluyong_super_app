# libs/

Shared, versioned code consumed across apps, services, platform modules, and AI services.

| Package | Purpose |
|---|---|
| `shared-ui/` | Shared design system components |
| `shared-contracts/` | OpenAPI specs and event schemas — the only sanctioned way modules interact across boundaries |
| `shared-utils/` | Shared utility code |

Changes to `shared-contracts/` are versioned and go through governed API and schema review — see Section 21 (Repository Structure) of the root README.
